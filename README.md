# BANK MARKETING PERDICTION (DECISION TREE CLASSIFIER)

This project predicts whether a customer will subscribe to a term deposit using a Decision Tree Classifier.
The dataset used is from the kaggle Bank Marketing dataset.



## 📌 PROJECT OVERVIEW:

This project uses a **Decision Tree Classifier** to predict whether a customer will subscribe to a term deposit based on demographic and behavioral data.
The dataset used is from the :contentReference[oaicite:0]{index=0} Bank Marketing dataset.
The goal is to analyze customer behavior, perform data visualization, and build a predictive model.



## 📊 FEATURES:

- Exploratory Data Analysis (EDA)
- Data Visualization (8 charts)
- Data preprocessing and encoding
- Decision Tree model training
- Model evaluation (accuracy, classification report)
- Decision tree visualization
